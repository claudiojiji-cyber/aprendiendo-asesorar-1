# Aprendiendo a Asesorar

## Descripción del Proyecto
Este proyecto es un sitio web educativo destinado a enseñar sobre tecnología, productos y ventas de manera clara y sencilla. Contiene información sobre cursos, productos y un formulario de contacto para que los usuarios puedan enviar sus comentarios.

## Estructura del Proyecto
- `index.html` : archivo principal con la estructura del sitio (header, nav, main, section, footer)
- `estilo.css` : archivo de estilos CSS para diseño y responsividad
- `IMAGENES/` : carpeta que contiene las imágenes de los cursos y productos
- `README.md` : descripción del proyecto

## Secciones del Sitio
1. **Inicio**: Bienvenida y breve introducción.
2. **Cursos / Productos**: Listado de cursos con imagen, nombre, SKU y precio.
3. **Contacto**: Formulario para que los usuarios dejen sus comentarios.
4. **Footer**: Información de contacto, redes sociales y ubicación.

## Tecnologías Utilizadas
- HTML5
- CSS3 
- Formulario 

## Adaptacion para varias pantallas
El sitio está adaptado para dispositivos móviles mediante media queries, asegurando que la navegación, las secciones y los formularios se visualicen correctamente en pantallas pequeñas.
